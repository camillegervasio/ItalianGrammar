# Italian Grammar Reference

This repository contains my personal Italian grammar reference book.

The document began as notes from classes, textbooks, conversation practice, immersion study in Cefalù, Sicily, and independent study. Over time it evolved into a structured reference manual organized for retrieval rather than by textbook chapter.

Topics include:

* Passato prossimo
* Imperfetto
* Congiuntivo
* Pronomi diretti e indiretti
* Pronomi doppi
* Ci e Ne
* Preposizioni
* Espressioni di tempo
* Verbi comuni e costruzioni idiomatiche

The goal is not to provide a complete grammar of Italian, but rather a practical reference for an adult English-speaking learner.

The repository contains both:

* A Pages document (editable source)
* A PDF version (easy to read on any platform)

If this document is useful to other learners, feel free to use it as inspiration for building your own personalized grammar reference.
